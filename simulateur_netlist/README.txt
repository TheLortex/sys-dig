Pour compiler le programme :
-lancer make all


Pour executer le programme :
-Lancer la commande "./simulateur_netlist file.net ouput" pour compiler la netlist file.net, et obtenir le résultat sous forme de fichier c++ output.cpp et executable outpu

Pour lancer le simulateur :
-Lancer la commande ./ouput (avec ouput le nom de l'executable créé comme précédemment)
-Suivre les indications données par la console.


Pour plus d'informations, se référer au rapport situer à la racine du dossier.
